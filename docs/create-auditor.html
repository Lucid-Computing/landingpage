The Container Standard

Build Custom Auditors

An Auditor is just a Container. Write your logic, push to Git, and let the platform handle the chain of custody.

View Contracts

1 Choose Persona

Select a template: Developer, Security, or Compliance.

2 Define Logic

Implement the check. Example: A "Crash Catcher" for Developers.

3 Containerize

Package the logic into a standard OCI image.

4 Configure Pipeline

Initialize the CI/CD workflow and set identity secrets.

5 Git Push (Auto-Sign)

The CI pipeline detects the push and signs the image automatically.

6 Publish & Chain

Push to registry. It is now ready to be referenced in a Mandate.
